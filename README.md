README: This respository contains a data pipeline I developed in Matlab and R for Dr. Chris Erb of the CMND Lab. The pipeline extract, preprocesses, and restructure raw data to a format appropriate for a Time Continuous Multiple Regression (TCMR) which was developed by Scherbaum and Dshemuchadse (2019)

TCMR_processing_pipeline.pptx contains diagram illustrating the steps of the pipeline from data extraction and pre-processing (CeanfileFlanker.R), data restructing (Convert2Scherbaum.m), then TCMR analysis (Analyze_demodata.m).
